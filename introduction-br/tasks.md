# 50 бизнес-задач на Go (по мотивам учебных)

## Блок 1. Онбординг и работа с клиентами (задачи 1–5)

1. **Приветствие клиента**  
   Напишите программу, которая при запуске спрашивает имя клиента и выводит «Добро пожаловать, X! Сегодня скидка 5%».

2. **Калькулятор возраста клиента**  
   Запросите год рождения, выведите возраст клиента и через сколько лет он станет пенсионером (условно: 65 лет).

3. **Тип клиента (аналог алиаса)**  
   Объявите тип `ClientID` на основе `string`. Создайте переменную и присвойте ей значение `"CUST-123"`.

4. **Игнорирование лишних полей в анкете**  
   При обработке формы клиент вводит: имя, email, телефон, возраст, пол. Используйте только имя и email, остальное — через `_`.

5. **Обмен валют в кассе**  
   У вас есть курс USD → EUR. Поменяйте местами значения переменных `usdRate` и `eurRate` без третьей переменной.

## 🧨 БОНУС (настоящая бизнес-задача)

### **Асинхронная обработка заказов (Pipeline)**

Постройте конвейер обработки заказа:

```
gen(заказы) → validate() → enrich(цены) → saveToDB()
```

- `gen` — генерирует ID заказов (100, 101, 102...)
- `validate` — проверяет, что ID > 100
- `enrich` — добавляет цену (например, map[id]price)
- `saveToDB` — "сохраняет" (просто печатает)

Каждый этап — своя горутина + каналы.
